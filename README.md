# MMWave Radar AI Project

This repository contains the complete implementation of an MMWave Radar AI system for metal object detection.

## üéØ Project Overview

This project demonstrates radar signal processing and AI-based classification using synthetic FMCW radar data. It achieves **99% classification accuracy** on metal vs. non-metal detection.

## üìÅ Project Structure

### **Part 1: Synthetic Radar Signal Processing** ‚úÖ
-   `radar_simulation.ipynb`: Generates synthetic FMCW radar signals and visualizes Range-Doppler heatmaps for Empty, Metal, and Clutter scenarios.

### **Part 2: Metal vs Non-Metal Classification** ‚úÖ
-   `classification_model.ipynb`: Trains a CNN to classify Metal vs Non-Metal with **99% test accuracy**.
-   `radar_cnn_model.pth`: Trained model weights (generated by notebook).
-   `radar_utils.py`: Shared utility library for radar signal generation and processing.

### **Part 3: Hidden Object Detection Challenge** ‚úÖ
-   `hidden_object_detection.ipynb`: Demonstrates background subtraction technique to detect hidden metal objects in clutter.

### **Part 4: Deployment Logic** ‚úÖ
-   `deployment_design.pdf`: Complete design document for real-time deployment.

### **Part 5: Demo Video** ‚è≥
-   `demo_video`: Recorded Video explaining approach, synthetic data generation, CNN model,etc.

## üìä Key Results

- **Classification Accuracy**: 99% on test set (200 samples)
- **Model Architecture**: Custom CNN with 3 conv layers + batch normalization
- **Dataset**: 1000 synthetic radar samples (balanced metal/non-metal)
- **Key Innovation**: Global normalization (120dB) instead of per-sample normalization

## üì¶ Core Components

### `radar_utils.py`
Shared utility library containing:
- `generate_radar_cube()`: FMCW radar signal simulation
- `process_radar_cube()`: Range-Doppler processing (2D FFT)
- `get_random_scenario()`: Scenario generation (metal, non-metal, clutter, hidden)

### Configuration
- **Frequency**: 77 GHz (automotive radar)
- **Bandwidth**: 4 GHz
- **Chirps**: 128 per frame
- **Samples**: 600 per chirp
